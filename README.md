# Make Insure - AI-Powered Insurance Management
*A project for the GitHub DevDays Hackathon 2026*

**Make Insure** is a full-stack unified platform designed to manage both vehicle and health insurance portfolios, utilizing AI for instant document auditing and fraud detection.

## 🚀 Microsoft & Open Source Tech Stack
*   **Frontend:** React.js, Tailwind CSS
*   **Backend:** Node.js, Express, MongoDB
*   **AI Integration:** Azure AI Document Intelligence (for automated RC and medical bill verification)
*   **Cloud Hosting:** Azure App Service
*   **DevOps:** GitHub Actions (CI/CD) & GitHub Copilot

*Note: Active development for Round 2 begins May 25, 2026.*
